💡 親愛的導師，您好！

這套系統能幫助您從傳統紙本或繁雜的 Excel 中解放。不論是學生的出缺席狀態（全日/半日）、多科目成績、多項每日作業、平時行規、突發狀況與心理諮商對談，皆能在手機或電腦上隨時秒級登錄與修改。

家長僅需憑「學號 + 專屬 6 位通行密碼」，就能即時看見其子弟的完整軌跡；系統更能一鍵調研所有在校紀錄，呼叫 Google 最新的 Gemini 3.5 Flash AI 智慧大腦，自動編製出溫馨、詳盡且具有親和力的發展輔導建議，並可直接輸出成精美的 A4 PDF 與列印報告！

部署前的準備清單：

  • 一個常用的 Google 帳號 (Gmail)
  • 約 5~10 分鐘的專注時間
  • 完全不需要任何網頁程式碼編寫基礎，跟著下方步驟「複製、貼上」即可！
第一步：建立試算表 第二步：申請 AI 金鑰 第三步：架設 GAS 網頁 📋 獲取完整安裝代碼

1 建立 Google 試算表雲端資料庫

Database Setup

我們將使用 Google 試算表做為免費、超快速讀寫且終身備份的個人雲端資料庫。

A

建立全新試算表

登入您的 Google 雲端硬碟，點選「新增」 -> 「Google 試算表」。

B

複製試算表網址識別碼 (Spreadsheet ID)

打開剛創立的試算表，觀察網址列。在網址中間，會有一串長長的亂碼。這串亂碼就是您的 SPREADSHEET_ID，請將它複製備用。

https://docs.google.com/spreadsheets/d/這一段長長的英數亂碼就是ID/edit#gid=0

2 申請 Google Gemini AI API 金鑰 (大腦設定)

Gemini Key Application

AI 輔導報告功能是藉由 Google 免費提供給個人與開發者的 API 服務。我們需要申請一把金鑰來驅動大腦。

A

造訪 Google AI Studio

使用同一個 Google 帳號，點選此連結前往申請頁面： Google AI Studio 官方網站。

B

獲取 API 金鑰 (API Key)

1. 點選左上角藍色的 "Get API Key"。
2. 點擊 "Create API Key" 並選擇您的新雲端專案，或建立新專案。
3. 生成成功後，會顯示一串以 AIzaSy... 開頭的長金鑰，點選 Copy 複製，並安全儲存在您個人的記事本中。

3 架設 Google Apps Script 網頁系統

Google Apps Script Web-App

Google Apps Script (GAS) 是將試算表轉換為專業網頁應用程式的官方免租金伺服器平台。

A

啟動 Apps Script 編輯面板

在您剛剛創立的 Google 試算表中，點選頂部功能列選單的 「延伸功能」 -> 「Apps Script」。

B

寫入 code.gs 後端主引擎

將預設的 Code.gs 檔案內容全部刪除清空。接著，在下方 第四步 的儲存箱中點擊「📋 複製此代碼」，將 code.gs 代碼完整貼入。

C

修改核心參數 (關鍵步驟)

在您貼入的 code.gs 最上方 1~5 行，修改以下參數：
1. SPREADSHEET_ID: 填入剛複製的試算表 ID。若您的 script 與試算表綁定，亦可保留空字串，系統會自適應取得。
2. GEMINI_API_KEY: 填入您剛剛複製的 AIzaSy... API金鑰。
3. ADMIN_PASSWORD: 教師管理端登入密碼，例如預設的 Admin888，或自行修改。

D

新增並編寫 Index.html 前端網頁

1. 在 Apps Script 左側邊欄檔案清單上方，點擊 「+」 號（新增檔案），並選擇 「HTML」。
2. 檔案名稱務必命名為 Index (大小寫需一致，不要加上 .html)。
3. 將預設內容全部清空，在下方 第四步 的儲存箱中點擊「📋 複製此代碼」，將 Index.html 代碼完整貼入，隨後點擊上方的 「儲存」 圖示。

E

正式發佈為網頁應用程式

1. 點擊右上角藍色的 「網頁部署」 -> 「新增部署」。
2. 點擊左側「選取類型」齒輪，選取 「網頁應用程式」。
3. 「執行身分」設定為：「您 (您的 Google 帳號)」。
4. 「誰有權限存取」設定為：「所有人」（這樣家長端及學生才能安全登入，不用授權）。
5. 點選 「部署」。
⚠️ 安全性提問：此時 Google 會彈出視窗提示「需要授權」。點擊「授予存取權」 -> 選您的帳號 -> 點選左下方灰色小字 「進階 (Advanced)」 -> 點選 「前往『未命名專案』(不安全)」 -> 點選 「允許 (Allow)」 即可。
6. 部署成功後，系統會生成一個 「網頁應用程式 URL」。這就是您的專屬系統網址！ 請複製並將其存入書籤，這也是提供給家長登入使用的網址。

⚙️ 1. 後端主要引擎 (code.gs) 程式代碼

建立在 Apps Script 中，替換 Code.gs 所有內容

🖥️ 2. 前端網頁介面 (Index.html) 網頁代碼

建立在 Apps Script 中，命名為 Index，替換其所有代碼

5 手冊日常使用指南

Operations Manual

👩‍🏫 導師日常管理操作

  1. 教師端登入： 輸入預設密碼 Admin888 (或您在 code.gs 設定的密鑰)。
  2. 建立學籍名冊： 點擊「新增學生」，為每位學生設定學號、姓名，並取得系統自動產生的 6 位家長安全通行碼。
  3. 登錄與修改紀錄： 在首頁登錄當天考驗分數（可按科目、單元動態增加）、登記作業狀態、出缺席等。若發現登錄錯誤，至「歷史軌跡」點擊「編輯」即可即時修改並覆寫雲端。
  4. 產生 AI 分析： 當累積一定紀錄時，在「AI 智能評估報告」分頁選擇適合的報告長度，點擊即可生成優雅的 Markdown 行為學業建議，並可隨時手動修改再按儲存！

👨‍👩‍👦 家長與學生端查閱

  1. 專屬連結登入： 家長在手機或電腦點擊同一網址（Web-App URL）。
  2. 雙重隱私驗證： 輸入導師配發的「班級學號」與「專屬 6 位通行碼」。由於不以姓名登入，降低了重名出錯率。
  3. 閱讀歷程軌跡： 即時看見學生的出席率與作業狀態統計，並查閱由導師修改發佈後的 AI 智能綜合發展評估，提供家庭陪伴方向。

💡 技術免責與安全宣告： 本系統所有學生歷程數據皆儲存在您自創的 Google 雲端帳戶下，AI 大腦串接採用安全 HTTPS 高規通訊，第三方絕無權限涉足，安全防護 100%。
